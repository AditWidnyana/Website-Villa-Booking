**#Website Villa Booking**

**Deskripsi**
Website Villa Booking merupakan project frontend berbasis web yang dibuat untuk menampilkan informasi villa, detail fasilitas, 
serta pemesanan villa secara online. Website ini dirancang dengan tampilan yang 
sederhana, informatif, dan responsif untuk memudahkan pengguna dalam melihat dan memilih villa yang tersedia.
